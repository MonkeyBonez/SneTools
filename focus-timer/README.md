# focus-timer

Minimal focused work timer.

## Usage

Open `timer.html` in any browser.

## Controls

| Action | How |
|---|---|
| Start | Play button |
| Pause | Pause button |
| Restart | Restart button (visible when paused) |
| Color cycle | Small dot — bottom right corner |

## Features

- Counts up in minutes
- Color cycle mode: theme shifts dark → light over 90-min cycles (toggle bottom-right dot)
- No dependencies — single HTML file, works offline
